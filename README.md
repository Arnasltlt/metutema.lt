# Met≈≥ Tema

A bilingual (Lithuanian/English) Next.js website explaining the "Theme of the Year" family tradition.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) - you'll be redirected to `/lt` (Lithuanian) by default.

4. To view the English version, navigate to [http://localhost:3000/en](http://localhost:3000/en)

## Deployment to Vercel

1. Push your code to a GitHub repository
2. Import the project in Vercel
3. Point your domain `metutema.com` to the Vercel deployment
4. The site will automatically build and deploy

## Features

- Bilingual support (Lithuanian default, English at `/en`)
- Celestial calendar aesthetic with dark theme and amber accents
- Responsive design
- Language switcher in the hero section
